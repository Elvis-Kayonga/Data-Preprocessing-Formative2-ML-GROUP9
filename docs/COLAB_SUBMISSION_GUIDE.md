# Colab Submission Guide

## What to upload to Google Colab

Upload these executed notebooks:
- `notebooks/Task1_Product_Recommendation.ipynb`
- `notebooks/Task2_Face_Image_Processing.ipynb`
- `notebooks/Task3_Voice_Audio_Processing.ipynb`
- `notebooks/Task4_Multimodal_Integration.ipynb`

Upload raw CSV files:
- `data/raw/customer_social_profiles.csv`
- `data/raw/customer_transactions.csv`

## Colab folder setup (recommended)

In Colab, create:

- `/content/data/raw/`
- `/content/data/processed/`
- `/content/models/`
- `/content/outputs/plots/`

Then run notebooks from top to bottom.

## Evidence to include in report

- Data cleaning checks (nulls before/after)
- Merge validation output
- EDA screenshots (distribution, outliers, correlation)
- Model comparison chart
- Confusion matrix and feature importance chart
- Multimodal gate simulation output

## Final submission checks

- Notebook cells are all executed
- No runtime errors shown in outputs
- Required artifacts are generated
- Notebook names follow Task1..Task4
